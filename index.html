AI Story Sculptor

AI Story Sculptor is a sophisticated web platform I developed from scratch, utilizing a full-stack approach. The project leverages Django for the backend, ensuring a secure and scalable environment, while the frontend is crafted using HTML, CSS, and JavaScript for a seamless and responsive user experience. The core functionality centers around integrating OpenAI's GPT-4o API, enabling dynamic AI-driven story generation.

The platform supports user registration and authentication, allowing users to create, customize, and manage personalized narratives. Key features include token-based transactions, automated chapter rendering, and API-driven custom cover art generation.

AI Story Sculptor also integrates with Stripe for secure payment processing, handling subscription models and one-time token purchases. The system is designed to be modular and scalable, making it easy to implement new features or adapt to different types of user-generated content.

Quiz Master

Quiz Master is a dynamic web application developed during my time at DevCodeCamp. This project is built using JavaScript, HTML, and CSS, with real-time quiz data sourced from the OpenTriviaDb API. The app allows users to create an account, providing them with access to a wide range of quizzes across different categories and difficulty levels.

The application architecture is based on asynchronous API calls to fetch and display quiz questions, enabling a seamless user experience. It features client-side JavaScript logic to manage game flow, including shuffling answer options, handling user input, and providing immediate feedback on user responses. The app uses DOM manipulation to dynamically update content without the need for page reloads, offering a more interactive and engaging user interface.

Crypto Insight

Crypto Insight is a powerful full-stack web application that helps users track and analyze cryptocurrency portfolios in real-time. Built using React/Vite on the front end and Django on the back end, it showcases modern development practices for secure user authentication, real-time data integration, and dynamic UI experiences.

The application fetches live crypto prices from external APIs, storing them in MongoDB for quick retrieval. Users can manage their own portfolios, view visual charts of performance, and take advantage of a sleek, dark-themed UI built with Tailwind CSS.

Project Overview
  • Name: Crypto Insight
  • Purpose: Provide real-time crypto price tracking, a user-managed portfolio, and visual charts to help investors monitor their holdings and overall portfolio value.
  • Key Features:
    • User Authentication: Secure login/registration with JWT authentication.
    • Portfolio Management: Add crypto holdings and see up-to-date valuations.
    • Price & Market Data: Fetches real-time crypto prices from Coingecko API.
    • Charts & Visuals: Uses Chart.js for dynamic crypto trend visualization.
    • Dark Theme UI: Built with Tailwind CSS for a sleek, modern interface.
Architecture & Technologies
  • Front End: React (Vite), Tailwind CSS, Chart.js, Axios.
  • Back End: Django, Django REST Framework, SimpleJWT, MongoDB.
  • Deployment & Tooling: Git, GitHub, Virtual Environments.
